# Portfolio Docs

Professional Docusaurus documentation site showcasing 3 open-source projects.

## Projects

1. **AI-Chag 2025** - AI-powered Video Retrieval with CLIP
2. **Ldx-Insight** - Open Data Integration Platform
3. **SmartCity Platform** - IoT with Tiered Storage & ML

## Quick Start

```bash
npm start
```

Visit http://localhost:3000

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

## Structure

- `/docs` - Documentation content
- `/src/pages` - Custom pages (homepage, about)
- `/static` - Static assets
